那些年，我用 Django Admin 接的案子

Language

Chinese talk w. English slides

Category

Web Frameworks

Python Level

Intermediate

Slides Link

http://www.slideshare.net/flywindy/django-admin-63246437

Abstract

強大的後台管理系統（Django Admin），是 Django 在眾多網站框架中脫穎而出的主因之一。
不僅擁有快速開發的特性，也保留了深度客製的彈性。
本次演講將透過接案的情境，分享如何針對不同需求，修改顯示欄位、排序、Filter、搜尋、權限管理、介面優化等需要客製化的項目。
只要學會了這些技巧，人人都可以用 Django Admin 接案！

Michelle Leu

Django Girls Taipei Organizer，致力於讓更多女孩愛上寫程式這件事。
目前在 GliaCloud 當個小小工程師和不專業泡茶師。
