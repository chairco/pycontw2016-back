用Google Cloud Platform玩交通資料分析

Language

Chinese talk w. Chinese slides

Category

Data Analysis

Python Level

Intermediate

Slides Link

https://docs.google.com/presentation/d/19AeaYxblhQ4lbZ_gReZAnBuOmzBu6UjXUWCV61DwV94/edit?usp=sharing

Abstract

一個資料的流程涉及資料抓取、清理轉換、儲存、選用模型分析，到資料視覺化或提供API接口等等，需要各種不同的技術整合，可能也分散在各種不同平台上；在這個演講中，會以國道ETC車流資料為例，介紹在Google Cloud Platform上利用AppEngine建立資料爬蟲，串連BigQuery儲存，最後應用Datalab分析，在雲端上完成一個資料分析的流程。

柯維然

穿上制服是個菜鳥警官，忙於勤務與警政工作；脫下制服是個熱愛用數據與程式解決問題的資料戰警；目前服務於新竹市警察局，與在交通大學資訊工程研究所在職進修中
