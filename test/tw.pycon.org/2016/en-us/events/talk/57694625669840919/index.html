如何打造關鍵字精靈

Language

Chinese talk w. Chinese slides

Category

Data Analysis

Python Level

Intermediate

Abstract

關鍵字的應用身處於每個人的日常生活及企業應用之中，舉凡欲尋找有關沖繩相關旅遊資訊該如何下出正確的關鍵字，
或者是你是個廣告業務想要投放嬰兒用品的廣告不知道該下哪些關鍵字。這時你會需要打造一個關鍵字精靈來協助你
演講內容會介紹 Text Mining 的一些基本觀念、打造關鍵字聯想精靈的技巧及所需要套件資源，最後會介紹一些實際的應用。

Description

* 初版投影片: https://docs.google.com/presentation/d/1zywCqwyn5YAMCaeCXfcRZpgp_5LQUNhqyXWiAyqn7II/edit?usp=sharing * 現在網頁文章資料量呈現爆炸的狀況，同樣的在 PIXNET 痞客邦目前擁有上億篇文章。面對這海量的文章我們該如何分析，並且知道不同事情(關鍵字)之間彼此的關連呢? * 這裡會介紹相關的工具及演算法，各位回去也可以自行收集資料打照屬於自己的關鍵字精靈，並且實際介紹如何運用在 PIXNET 及個人生活上 * 工具介紹 : python , spark , jieba , word2vec , google search console
