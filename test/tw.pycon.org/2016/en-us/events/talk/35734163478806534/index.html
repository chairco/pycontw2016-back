用Numpy做一個自己的股票分析系統

Language

Chinese talk w. Chinese slides

Category

Python Libraries

Python Level

Intermediate

Abstract

在大數據時代，股票的技術分析技術開始爆發，各種各樣的參數與分析技術紛紛出籠，市售軟體通常有其挶限性，不能做到完全符合自身需要的技術線，Python提供了良好的資料分析平台，透過Numpy與Amcharts.js等套件的結合，讓使用者能夠完全控制自己的分析策略。本分享為一個股票技術分析的敲門磚，讓使用者輕鬆建構一個屬於自己的股票分析系統。

Description
