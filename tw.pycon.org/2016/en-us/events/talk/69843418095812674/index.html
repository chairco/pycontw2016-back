連淡水阿嬤都聽得懂的機器學習套件入門 scikit-learn

Language

Chinese talk w. Chinese slides

Category

Science

Python Level

Intermediate

Slides Link

http://www.slideshare.net/aacs0130/scikitlearn-62706630

Abstract

機器學習 Machine Learning 是最近相當熱門的領域，隨著越來越多機器學習套件的推出，使得機器學習更加平易近人，更多人可以享受到機器學習帶來的好處。本演講將會以 Python 機器學習套件 scikit-learn 為例，介紹機器學習的訓練流程以及如何使用 scikit-learn 分析預測資料，包含1. 前處理 preprocessing, 2. 特徵值抽取 feature extraction, 3. 選擇機器學習演算法, 4. 訓練模型，5. 調參數 tune parameters，6. 預測結果等等。本演講不會介紹Deep Learning, 很多機器學習演算法跟演算法背後的數學意義。scikit-learn 能讓淡水阿嬤都能簡單使用機器學習分析總統選情，阿孫要先教會阿嬤 Python 語法跟安裝 scikit-learn 套件唷，科科。

Cicilia Chia-ying Lee

Pythoner, data scientist, NLP engineer.
