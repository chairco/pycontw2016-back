摘要

色彩管理目前被廣泛使用在作業系統/影像編修程式/廣告、美術、印刷、顯示產業等，重要性如同空氣一樣，我們每天接觸但卻沒有真正意識到它的存在。 色彩管理是甚麼 ? 如何將色彩管理程式化 ? 如何(適合)用 python 來實作一個麼?
我們來瞧瞧 !
* [補習] 將會介紹色彩定義/管理/轉換的知識
* [仲介] 介紹 python 的色彩管理套件 (Pillow.ImageCms based on pyCMS)
* [我是builder我自ㄍㄢˋ] side-project架構

CAUTION - 此演講不會有詳細 python 技術探討, 而是一位想用 python 自刻一套色彩管理系統, 進而瞭解整個色彩管理系統的宅宅的研究心得.

說明

當你透過相機的觀景窗拍了張照片, 你認為這張照片的顏色就像若瑄一樣正, 結果將相片存入電腦時用小畫家打開來看卻發現顏色歪成雪芙(還是很正啊~ 只是哪裡怪怪的..) 當你看到雜誌上的美食栩栩如生令人食指大動, 用掃描機掃下來/手機拍起來, 傳給好朋友約下次一起去品嚐, 朋友卻告訴你, 看起來讓他/她沒有胃口, What's going wrong ? 每個人感知的顏色都略有不同, 如何定義出一套包含所有可能的顏色的色彩系統, 甚至讓不同色彩空間的顏色可以交互轉換, 是門相當基礎但重要的學問. 想知道更多 (http://content.teldap.tw/index/blog/?p=1640) 所以會 1. 介紹顏色從何何來 2. CIE-RGB如何出現, 如何從CIE-RGB轉變成CIE-XYZ(虛擬顏色空間), 3. 如何讓各種顏色空間在不同設備轉換而達到一致的顏色效果 (ICC Profile & Color management system) 4. 介紹 Pillow.ImageCms based on pyCMS 5. Side project 當前的進度 & demo 色彩轉換 *Slide - https://goo.gl/DOZ92g
