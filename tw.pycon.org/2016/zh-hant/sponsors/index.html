贊助夥伴

PARTNER

GliaCloud

GliaCloud - 集雅科技，成立於 2015 年，專注於人工智慧與大資料應用解決方案。希望藉由資料處理平台降低大資料分析的技術門檻，並結合人工智慧，讓企業與一般人不是只產生資料，而有能力處理且分析資料，運用學習與預測機制到各種商業或生活層面，真正享受到大資料帶來的好處。

中央研究院資訊科學研究所

中央研究院資訊科學研究所於1977年開始設立籌備處，歷經五年籌備， 於1982年9月正式成立研究所，是中央研究院數 理組十一個單位之一。目前編制內有39位研究人員，另外有29位博士後研究學者，將近300位專任之資訊技術人員與非全時之研究助理，支援資訊領域之研究與系統之開發。「件件工作，反映自我，凡經我手，必為佳作」是全體同仁一致秉持的工作信念，重視工作之卓越品質，發揮最佳綜效之團隊精神。

開放文化基金會

開放文化基金會成立於 2014 年，由多個開源社群及活動組織共同發起，希望協助台灣蓬勃的開放社群，進一步促進開放源碼、開放資料、開放硬體等自由精神，能更廣泛的於各領域中應用，同時也協助政府、企業及非政府組織更能了解開源軟體的優勢、開放資料的重要，進而推廣開放協作的文化。 2015 年起我們除了支援本地社群主辦多場中大型活動，也和英國代表處、美國在台協會及世界銀行等單位共同舉辦各種主題講座，深耕台灣的開放文化並與國際交流。

玉山銀行

以台灣最高的山為名的玉山銀行，成立於1992年，秉持著「心清如玉，義重如山」的理念，期望成為金融業界的服務典範。近年來在金融創新發展部分，玉山銀行創造了各種不同的新型態金融服務，像是跨產業合作的第三方支付服務等。在未來，玉山銀行將持續推展全面性的金融服務以創造不可取代的地位，並由衷的希望能夠透過團隊的合作與努力登上國際金融舞台，期待透過玉山清新、專業的團隊，成就台灣的玉山、世界的玉山。

Akatsuki 曉數碼

Akatsuki創立於2010年6月,開發許多款熱門手機遊戲《萬千回憶 (Thousand Memories)》，與BANDAI NAMCO Studio共同開發的《Tales of Link》, 《七龍珠Z 爆裂激戰》也頗受好評。 2014年Akatsuki成立台灣分公司，這也是Akatsuki首度跨出日本在海外成立事業據點，歡迎遊戲界設計、企劃、製作等等人才加入台灣的第一期創設成員。

Appier

Appier是專注於人工智慧與跨螢技術的新創公司，以人工智慧技術為基礎幫助企業解決跨螢世代的複雜難題，並讓更多領域的企業透過人工智慧應用擬定最佳商業策略。Appier成立於2012年，由充滿熱情的電腦、資訊科學家及網路行銷專家組成，成員來自Google、Intel、Yahoo等知名企業，技術團隊來自擁有豐富人工智慧研究成果的哈佛及史丹佛大學。Appier總部設於台北，結合遍佈亞洲11個城市的營運據點，為超過500家知名品牌與代理商提供優質服務。

Carousell

Carousell 是目前東南亞最大，且成長最快速的行動交易社群 App，簡單的操作讓使用者拍賣物品 像拍照一樣簡單。被列為 Google Play 上全球熱門應用程式，並在新加坡生活風格及購物類應用程式排名雙冠，Carousell 已經成為年輕族群網路購物的重要據點。利用 Carousell，使用者可以買賣不論是二手或全新的物品。透過交換而不是丟棄的方式，環保愛地球

Gandi

Gandi 提供網域名稱註冊與雲端伺服器服務，我們提供超過600種後綴，並持續增加新的域名種類。Gandi 的新服務 LiveDNS 可讓您利用程式編輯域名的 DNS 資料並即時更新系統，使用 LiveDNS 不需費用。詳細資訊請上 https://www.gandi.net/

MYSQL

MySQL是當今全世界最受歡迎的開放源碼資料庫。它的性能、可靠度和易用性都經是許多使用者長久的長考驗，MySQL己成為網路為基礎的應用系 統的首選資料庫包括如臉書、推特、Youtub、雅虎等許多著名大型用戶都以MySQL為主要資料庫。甲骨文自從擁有MySQL的智財權以來引領 MySQL的創新，開發出許多的新功能推動新一代網路、雲、行動和嵌入式應用系統的發展。想了解更多相關資訊請參考 http://www.mysql.com/

Microsoft Taiwan

Visual Studio Community: 完全免費的開發工具，並完全整合 Xamarin、Cordova 和 Unity 讓開發人員輕鬆開發 Android, iOS 和 Windows 跨平台應用程式，立即了解 >> http://aka.ms/vscommunity-zht Visual Studio Code: 完全免費且可在任何您熟悉平台 (Linux、OSX 和 Windows) 上使用的開發工具，並已整合 ASP.NET 5 和 Node.js 應用程式的偵錯，立即了解 >> http://aka.ms/vscodetw

Ruckus Wireless

Ruckus Wireless 是無線基礎架構市場的領導廠商，協助電信業者及企業領先因應市場急切所需的高頻寬應用及服務需求。Ruckus Smart Wi-Fi 技術改寫了無線網路的彈性、可靠性及價格適宜性。自從 2004 年開始營運以來，我們一直特別著重在無線應用的普及效能，並持續在效能創新方面領先業界。我們從不倚賴現有的參考設計射頻技術，因為它們無法實現高效能無線所需的容量、涵蓋範圍或干擾消除。我們組成了一支在全世界對無線技術擁有胸懷壯志的團隊，推出實現我們普及效能願景的創新解決方案。

Sudo 速度資訊

Sudo 旨在打造軟體工程師專屬的求職平台、提供求職者最高規格的媒合服務，協助人才找到最佳舞台盡情展現！ 透過線下活動：贊助各大技術年會如 PHP conference、JCConf、舉辦開發者交流社群-Android Taipei & Designer night、產業交流媒合會、校園職涯講座、WeStudent 實習平台，多管齊下協助軟體工程師找到最適性的工作，盡情發揮個人價值。 合作夥伴遍及矽谷、東京、香港及新加坡，藉此提供軟體專才及網路菁英多樣的優質機會；誓言要讓台灣重回世界舞台，以亞洲矽谷的身份聚焦國際目光。

銅級

IBM

IBM是具全球規模的認知運算及雲端服務公司，創新是IBM策略的核心。IBM的雲端開發平台 Bluemix 以Cloud Foundry 開放程式碼技術為建置基礎，在Bluemix 上可在iOS、Android 和HTML中使用JavaScript 來開發行動式應用程式: 對於Web 應用程式，可以使用Java、Ruby、Pyson、和Node .js 等語言。在IBM Bluemix中還能使用IBM獨有的物聯網IoT 開發模組，大數據 Waston Analytics 開發模組，Cloudant資料庫等最先進的軟體產品。

Lilee System 理立系統

LILEE Systems 來自美國，為火車、地鐵與巴士客戶提供完整的無線通訊解決方案，2012年更獲得美國 Internet Telephony 雜誌頒發 ”Product of the Year” 獎項。LILEE Systems的軟體定義無線電 (Software-Defined Radio) 技術，透過軟體調整無線電頻率及發射機輸出特性，提供更好的性能與有效通訊。此外，LILEE研發、量產的一系列車載無線通訊設備能夠提供無線IP，在 220 MHz、3G/4G 和Wi-Fi 間無縫切換，進一步提供乘客上網、安全監控和營運管理的加值應用。

Transtep Technology Inc.

龍雲數位整合公司為華鴻創投所投資，並由一群10年以上經驗的資深Linux技術團隊組成，專門解決各企業的雲端應用服務的整合，目前已成為各大SI系統整合商重要合作夥伴之一。主要服務為從雲服務到雲平台的商務整合與開發。經營團隊擁有豐富專案建置與系統研發經驗並高度重視技術能量，致力於提供各類型客戶最高品質技術服務平台與整合。

特別贊助

Eldarion, Inc.

自2009年以來 Eldarion 團隊一直在幫助人們加速從想法到發布產品的過程。我們為世界各地的客戶開發工具、app 和網站，我們客戶從成熟企業到想創業的個人都有。開源是Eldarion 的核心。我們的團隊成員領導或貢獻了一系列專案，包括 Python 、 Django 、 Pinax ，許多單獨的 Django app 。我們不只使用最好的工具來幫助客戶，我們更協助打造這些工具。

琅茶

喝好茶，是一趟回不去的味覺旅程。 琅茶每季走訪產地，嚴選上質茶園在最適天氣時辰，由優秀茶師採製的「單品茶」 (Single Origin)。不混茶，為每支「單品茶」編號，詳載茶事，與時俱進，售完不再，真真切切感受每一片山林不同時節的純粹美好。 一起品遍千變萬化的台灣茶吧！開展好茶地圖，認識自己的味蕾，找到屬於你「琅琅上口」的好茶。
